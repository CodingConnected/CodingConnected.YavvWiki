---
title: "Analyse: groentijden"
date: 2021-01-08
---

Deze analyse zoekt de tijd van start tot einde groen. Indien is ingesteld dat alleen primair groen moet worden gezocht kan dit worden aangevinkt bij de instellingen. In dat geval wordt _op einde groen_ gekeken of het alternatief realiseren bit (BIT7, dus het 8e significante bit in het GUS bericht) op staat. Een realisatie die halverwege wisselt van alternatief naar primair telt dus altijd mee.
